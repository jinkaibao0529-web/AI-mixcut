# V1 Acceptance Checklist

Use this checklist with real ecommerce materials before marking V1 release-ready.

## Test Material Set

- [ ] 3 finished videos with voiceover and optional BGM
- [ ] 5 product or benefit shots
- [ ] 1 BGM file and 1 BGM folder import sample
- [ ] 1 TTS voiceover generated by provider or placeholder path
- [ ] 1 subtitle preset, preferably `.ass` or `.json`

## Mode A: Finished-Video Remix

- [ ] Import finished videos as `finished_video`
- [ ] Confirm ASR and semantic slicing complete
- [ ] Confirm segments have semantic tags and position tags
- [ ] Export selected segments
- [ ] Export by semantic, selling-point, or visual tag folder
- [ ] Generate AI remix timeline
- [ ] Preview full timeline
- [ ] Fine-tune clip in/out points
- [ ] Split a segment and confirm replacement candidates still work
- [ ] Replace one timeline clip and confirm selection reason persists after reload
- [ ] Create one remove-BGM task and confirm it appears in AI Worker task list

## Mode B: Product-Shot Generation

- [ ] Import product or benefit shots as `product_shot` / `benefit_shot`
- [ ] Run batch visual tagging
- [ ] Confirm visual description, selling-point tags, and visual tags appear
- [ ] Correct one clip position with `开头 / 中间 / 结尾`
- [ ] Generate a script from manual input
- [ ] Generate a script from Douyin-derived source text
- [ ] Filter scripts by source and keyword
- [ ] Generate or select TTS voice asset
- [ ] Generate timeline from script semantics
- [ ] Confirm each clip shows a clear selection reason
- [ ] Replace one clip and confirm candidate score/reason display is useful

## Audio

- [ ] Import BGM
- [ ] Import or generate voice asset
- [ ] Analyze loudness for BGM and voice
- [ ] Confirm LUFS values appear in voice/BGM page
- [ ] Confirm LUFS values appear in timeline export panel
- [ ] Apply recommended audio policy
- [ ] Export with original audio kept
- [ ] Export with original audio removed
- [ ] Export with voice replacement
- [ ] Export with BGM overlay
- [ ] Export with loudness normalization enabled
- [ ] Listen for clipping, voice masking, and large loudness jumps

## Subtitles

- [ ] Import subtitle preset
- [ ] Edit subtitle style metadata
- [ ] Select subtitle preset in timeline export
- [ ] Enable burn-in subtitles
- [ ] Export and confirm subtitle timing is acceptable at sentence level
- [ ] Confirm font, size, outline, shadow, position, and margin match preset

## Edge Cases

- [ ] Video with no audio track
- [ ] Very short clip under 1 second
- [ ] Segment with missing semantic tag
- [ ] Product shot with no obvious selling point
- [ ] TTS provider failure or placeholder voice path
- [ ] Missing BGM asset after selection
- [ ] Missing subtitle preset after selection
- [ ] Source video deleted or moved

## Release Gate

- [ ] `./scripts/v1_smoke.sh` passes
- [ ] One complete Mode A export is playable
- [ ] One complete Mode B export is playable
- [ ] Exported audio is acceptable on phone speakers and headphones
- [ ] Burned subtitles are readable on mobile
- [ ] Known limitations are documented for AI Worker features
