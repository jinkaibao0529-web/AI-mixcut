# V1 验收清单

在确认 V1 可发布之前，请用真实电商素材按这份清单逐项验收。

## 测试素材准备

- [ ] 3 个带口播的成片视频，最好包含有 BGM 和无 BGM 两种情况
- [ ] 5 个产品展示或权益卖点镜头
- [ ] 1 个单独 BGM 文件
- [ ] 1 个 BGM 文件夹导入样本
- [ ] 1 条 TTS 生成的配音，或 TTS 占位音频
- [ ] 1 个字幕预设，优先使用 `.ass` 或 `.json`

## 模式 A：成片混剪

- [ ] 以 `finished_video` 类型导入成片素材
- [ ] 确认 ASR 和语义切分完成
- [ ] 确认片段有语义 tag 和位置 tag
- [ ] 导出勾选片段
- [ ] 按语义 tag、卖点 tag 或画面 tag 导出文件夹
- [ ] 生成 AI 混剪时间线
- [ ] 预览完整时间线
- [ ] 微调片段入点和出点
- [ ] 拆分一个片段，并确认替换候选仍可用
- [ ] 替换一个时间线片段，并确认重新打开后选择理由仍保留
- [ ] 创建一个“一键消除 BGM”任务，并确认它出现在 AI Worker 任务列表

## 模式 B：产品素材生成

- [ ] 以 `product_shot` 或 `benefit_shot` 类型导入产品/权益镜头
- [ ] 执行批量视觉打标
- [ ] 确认生成画面描述、卖点 tag、画面 tag
- [ ] 用 `开头 / 中间 / 结尾` 快速校正一个镜头位置
- [ ] 用手动输入生成一版文案
- [ ] 用抖音来源文本生成一版结构改写文案
- [ ] 按来源和关键词筛选文案
- [ ] 生成或选择一个 TTS 配音素材
- [ ] 按文案语义生成时间线
- [ ] 确认每个时间线片段都展示清晰的选择理由
- [ ] 替换一个片段，并确认候选评分/理由对判断有帮助

## 音频

- [ ] 导入 BGM
- [ ] 导入或生成配音素材
- [ ] 分析 BGM 和配音的响度
- [ ] 确认 LUFS 数值显示在“配音/BGM”页面
- [ ] 确认 LUFS 数值显示在时间线导出面板
- [ ] 应用推荐音频策略
- [ ] 导出时保留原音频
- [ ] 导出时移除原音频
- [ ] 导出时用配音替换原音频
- [ ] 导出时叠加 BGM
- [ ] 导出时开启统一响度
- [ ] 听检是否有爆音、口播被 BGM 盖住、音量忽大忽小等问题

## 字幕

- [ ] 导入字幕预设
- [ ] 编辑字幕样式 metadata
- [ ] 在时间线导出里选择字幕预设
- [ ] 开启字幕烧录
- [ ] 导出后确认句子级字幕时间基本可接受
- [ ] 确认字体、字号、描边、阴影、位置、边距符合预设

## 异常场景

- [ ] 无音轨视频
- [ ] 1 秒以内的超短片段
- [ ] 缺少语义 tag 的片段
- [ ] 没有明显卖点的产品镜头
- [ ] TTS Provider 失败或走占位配音路径
- [ ] 已选择的 BGM 素材缺失
- [ ] 已选择的字幕预设缺失
- [ ] 源视频被删除或移动

## 发布门槛

- [ ] `./scripts/v1_smoke.sh` 通过
- [ ] 模式 A 完整导出一个可播放视频
- [ ] 模式 B 完整导出一个可播放视频
- [ ] 导出音频在手机外放和耳机里都可接受
- [ ] 烧录字幕在手机竖屏观看时清晰可读
- [ ] AI Worker 相关限制已记录清楚
